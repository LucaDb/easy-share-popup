# Easy-share-popup

## Usage

### Install module
```
npm install easy-share-popup --save-dev
```

### Javascript

```
// ES6
import Share from 'easy-share-popup';

// browserify 
var Share = require('easy-share-popup');

var share = new Share('http://s2paganini.com');
share.plus();
share.facebook();
```

### Support 

* Facebook
* Google Plus
* Twitter
* Pinterest
* Tumblr
* Weibo
* Renren
